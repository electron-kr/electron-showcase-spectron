# electron-spectron-boilerplate

> Show how to use spectron with ava on electron.


## Dev

```
$ npm install
```

### Test with spectron

```
$ npm test
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
